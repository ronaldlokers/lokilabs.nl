Browser app that drives a Dreaver Motion One walking treadmill over
**Web Bluetooth**. No account, no backend — everything lives in the browser.

- Walk a survey-exact IAAF 400m athletics track: 2D lap ring or first-person
  3D scenic walk with a day/night cycle driven by distance
- Heart-rate-steered workouts: connect any BLE HR source and the app nudges
  belt speed to hold a target zone
- Weekly statistics dashboard, one-tap Strava upload, Withings weigh-in sync
- CI, E2E tests, and deploys via GitHub Actions

Built with Claude Code.
